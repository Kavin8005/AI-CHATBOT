# AI Study Assistant using Endee (RAG System)

## 🚀 Overview
This project is a Retrieval-Augmented Generation (RAG) based AI assistant that allows users to query their study materials.

## 🧠 Features
- Semantic Search using Endee Vector DB
- RAG-based Question Answering
- FastAPI backend

## 🏗️ Architecture
User Query → Embedding → Endee Search → Context Retrieval → LLM → Answer

## 🛠️ Tech Stack
- Python
- Endee Vector Database
- Sentence Transformers
- OpenAI API
- FastAPI

## ⚙️ Setup
1. Clone repo
2. Install dependencies
3. Run ingest.py
4. Start server

## 📌 Use Case
Helps students quickly understand notes using AI.

## 🔗 Endee Usage
Endee is used as the vector database for storing embeddings and retrieving context efficiently.
